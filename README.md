# 🏋 WhatYouNeed 

- 운동 커뮤니티 운영 및 운동 용품 판매 쇼핑몰
- Kafka, Zookeeper 를 사용한 실시간 채팅 기능
---

### 🔍 주요 기능
```
- 회원 : 로그인 | 회원가입 | 소셜 로그인(구글, 카카오, 네이버) | 주문 조회/취소
- 채팅 : 상담원 채팅 | 문의사항 | 운동 피드 공유
- 상품 : 헤더 전체 검색 | 장바구니/바로주문 | 추천 상품 메인 진열
- 기타 : 본사 위치 안내
- 관리자 : 상품 등록/조회/검색/수정/삭제
```
